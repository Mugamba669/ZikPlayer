## Features

- 10 Graphic Visualisers
- Fetch online lyrics
- Edit Audio tags
- Download UG music
- Access offline lyrics if present
- Full Screen Graphical display
- 4Band Equalizer
- Audio Compressor (For Amplifying and Lowering the Bass)
- Room Effects (Scene, Auditronim,...)
