Key Function
Press "U" This function enables the user to open the file-system and load one track to the player.
Press "P" This function enables the user to activate theplaylist section.
In here you can load any folder containing musi tracks you would like to listen to.
Press "ArrowRight" This function loads the previous track * Only supported in playlist section*
Press "L" This function displays songs lyrics * Only supported when online*
Press "ArrowLeft" This function loads the next track Only supported in playlist section
Press "1" This takes you to the Home screen
Press "2" This takes you to the Equalizer section
Press "3" This takes you to the Tuning Section
Press "4" This takes you to the Compressor section
Press "5" This takes you to the Compressor section
